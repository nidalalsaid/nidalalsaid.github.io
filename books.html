Selected Books
Advanced Deep Learning Architectures & Techniques
Anurag Shrivastava · Betshrine Rachel Jibinsingh · Nidal Al Said · IIP Books
Machine learning and deep learning have rapidly evolved, creating vast opportunities for engineering students but lacking unified, structured study material that covers core concepts, applications, and emerging research. This book aims to bridge that gap by presenting foundational ideas alongside real-world applications, empowering learners to build practical ML solutions across diverse domains.

INTELLIGENT WEB MINING: DISCOVERING PATTERNS IN ONLINE DATA
Dr. Nidal Al Said
This book offers a structured, comprehensive introduction to web mining by covering its three core pillars—web content, structure, and usage mining—along with modern tools and frameworks. It provides an integrated perspective on analyzing user behavior, hyperlink patterns, and social media data, while also addressing scalable mining approaches in cloud environments.
